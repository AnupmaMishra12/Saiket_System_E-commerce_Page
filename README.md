# Saiket_System_E-commerce_Page
 This project is a responsive E-commerce web page named Shopcart, built using HTML, CSS, and JavaScript. It provides a modern shopping interface with product listings, filters, and cart functionality. The design focuses on simplicity, responsiveness, and user-friendly navigation.
 
##Features
Top Bar with contact info and promotional offers

Navigation Bar with categories, deals, new arrivals, and delivery options

Search functionality for products

Hero Section highlighting special offers with product image and CTA button

Filter buttons (Headphone Type, Price, Review, Color, Material, Offer)

Product Cards with image, name, price, and Add-to-Cart button

Responsive layout using CSS Grid and Flexbox.

Tech Stack

HTML5 → Page structure

CSS3 → Styling and responsiveness

JavaScript (basic) → Interactivity (cart buttons, search, etc.)
